# ❤️ DonViva — AI Powered Blood Donation Platform

DonViva is an intelligent cross-platform blood donation application built using Flutter and Firebase, designed to connect blood donors and recipients in real time.  

The platform enables users to register as blood donors, request blood during emergencies, and instantly connect with nearby compatible donors through a smart emergency alert system.

---

## 🚀 Features

### 🩸 Blood Donation Enrollment
- Users can register as active blood donors
- Maintain donor profiles and blood group information
- Real-time availability management

### 🆘 Emergency Blood Request System
- Request blood instantly during emergencies
- Sends emergency notifications to compatible nearby donors
- Fast donor-recipient connection workflow

### 🤖 AI Eligibility Checker
- AI-powered donor eligibility verification
- Helps determine whether a user is fit for blood donation
- Improves donor safety and screening efficiency

### 💬 AI Chatbot Assistant
- Integrated AI assistant for user guidance
- Answers common blood donation questions
- Assists users throughout the donation process

### 🔔 Real-Time Alert Notifications
- Instant SOS alerts for urgent blood requirements
- Real-time communication system
- Helps improve emergency response time

### ☁️ Firebase Integration
- Real-time database synchronization
- Secure authentication and backend services
- Cloud-based data management

## 🎯 Project Goal

The goal of DonViva is to simplify and modernize the blood donation ecosystem by combining:
- Real-time donor matching
- Emergency response systems
- AI-assisted healthcare support
- Secure cloud infrastructure

The application focuses on reducing delays during critical situations and improving accessibility to blood donors.
